【ステップ1】Eclipseでプロジェクトをクリーン&ビルド

1. Package Explorerで「UdemyKamedaJSPServlet」を選択
2. メニュー: 「Project」→「Clean...」
3. 「UdemyKamedaJSPServlet」を選択 → 「Clean」をクリック
4. プロジェクトを右クリック → 「Build Project」
   （または「Project」→「Build Project」）

【ステップ2】ビルド結果を確認

1. Package Explorerで以下のパスを展開：
   UdemyKamedaJSPServlet
   └─ build
       └─ classes
           └─ example
               ├─ Exercise501_1.class  ← これらが存在するか確認
               ├─ Exercise501_2.class
               ├─ Exercise501_3.class
               └─ Exercise501_4.class

2. これらの.classファイルが存在することを確認

【ステップ3】Tomcatサーバーを完全にクリーン

1. Serversビューで「localhost の Tomcat11 (Java25)」を右クリック
2. 「Clean...」をクリック
3. 「Clean Tomcat Work Directory...」をクリック
4. 「OK」をクリック

【ステップ4】Tomcatサーバーを停止（起動中の場合）

1. サーバーが起動中なら「Stop」で停止
2. 完全に停止するまで待つ（灰色のアイコンになる）

【ステップ5】Tomcatサーバーを起動

1. Serversビューで「localhost の Tomcat11 (Java25)」を右クリック
2. 「Start」をクリック
3. 起動ログをコンソールビューで確認

【ステップ6】ブラウザでアクセス

正常に起動したら、以下のURLでアクセス：

http://localhost:8080/UdemyKamedaJSPServlet/Exercise501_1

その後、リンクをクリックして Exercise501_3 にアクセスできるか確認

【方法1】ブラウザのキャッシュをクリア

1. ブラウザで Cmd + Shift + Delete (macOS) を押す
2. キャッシュをクリア
3. ページをリロード: Cmd + Shift + R

【方法2】Eclipseを完全に再起動

1. Eclipseを完全に終了
2. 再度起動
3. プロジェクトを開く
4. 上記のステップ1-6を再度実行

【方法3】サーバーを完全に再作成

1. Serversビューで「localhost の Tomcat11 (Java25)」を削除
2. Serversビューを右クリック → 「New」→「Server」
3. 「Apache」→「Tomcat v11.0 Server」を選択
4. 「Server runtime」で「Tomcat11 (Java25)」を選択
5. 「Finish」をクリック
6. プロジェクトを追加（Add and Remove...）
7. サーバーを起動
